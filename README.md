# Portfolio Website

A responsive personal portfolio website built with **Next.js** and **Tailwind CSS**, showcasing projects, skills, and contact information.  

## Features
- Fully responsive design for mobile, tablet, and desktop
- Smooth navigation and modern UI/UX
- Easy to customize with your own content and styles

## Tech Stack
- **Next.js** – React framework for server-side rendering
- **Tailwind CSS** – Utility-first CSS framework
- **Vercel** – Deployment platform for live hosting
